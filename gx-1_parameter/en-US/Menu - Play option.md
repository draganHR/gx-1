# Play option

Here you can specify how the pedals will work during performance.

| Parameter | Value | Explanation |
| --- | --- | --- |
| DOWN&UP SW | Specifies the function used when DOWN and UP are pressed at the same time. | |
| OFF | No function is controlled. |
| TUNER | Turns the tuner on/off. |
| LOOP | Turns the LOOP on/off. |
| MEM+1 | Switches to the next memory. |
| MEM-1 | Switches to the previous memory. |
| UP＆CTL1 SW | Specifies the function used when UP and CTL1 are pressed at the same time. | |
| OFF | No function is controlled. |
| TUNER | Turns the tuner on/off. |
| LOOP | Turns the LOOP on/off. |
| MEM+1 | Switches to the next memory. |
| MEM-1 | Switches to the previous memory. |
| MEM EXTENT MIN | U01–U99  P01–P99 | Sets the minimum value for the selectable memories. |
| MEM EXTENT MAX | Sets the maximum value for the selectable memories. |
| MEM CHANGE BST | OFF, ON | Sets whether to enable or disable memory switch boost when you long-press the [É] [Ç] switches. |

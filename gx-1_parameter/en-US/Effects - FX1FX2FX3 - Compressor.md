# Compressor

![](../Images/png/91951371__Web.png)![](../Images/png/91832843__Web.png)HEXARAY: Reduction meter

This is an effect that produces a long sustain by evening out the volume level of the input signal.

| Parameter | Value | Explanation |
| --- | --- | --- |
| ON/OFF | OFF, ON | Turns this effect on/off. |
| TYPE | BOSS COMP | This models a BOSS CS-3. |
|  | D-COMP | This models a MXR Dyna Comp. |
|  | ORANGE | This is modeled on the sound of the Dan Armstrong ORANGE SQUEEZER. |
|  | X-COMP | This uses MDP (Multi-Dimensional Processing) to obtain a consistently natural playing feel and sound that responds to the pitch range and dynamics of your phrase. |
|  | STEREO | This selects a stereo compressor. |
| SUSTAIN | 0–100 | Adjusts the range (time) over which low-level signals are boosted. Larger values will result in longer sustain. |
| ATTACK | 0–100 | Adjusts the strength of the picking attack when the strings are played. |
| LEVEL | 0–100 | Adjusts the volume. |

# Glitch

![](../Images/png/91951371__Web.png)

Creates a machine gun-like delay effect.

| Parameter | Value | Explanation |
| --- | --- | --- |
| ON/OFF | OFF, ON | Turns this effect on/off. |
| TRIGGER | OFF, ON | If this is ON, the GLITCH effect is applied. |
| TIME | 0–100 | Adjusts the length of the effect sound. |
| GLITCH | 0–100 | Adjusts the intensity of the effect. |
| BALANCE | 0–100 | Adjusts the volume balance between the direct sound and effect sound. A setting of “100” mutes the direct sound. |

# Bluetooth®

This configures the Bluetooth settings.

| Parameter | Value | Explanation |
| --- | --- | --- |
| BT SW | OFF, ON | Turns Bluetooth on/off. Turn this on to use Bluetooth. |
| BT ID | 1–10 | Switches between Bluetooth ID numbers. This makes it easier to find devices when connecting. |

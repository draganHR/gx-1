# Fv (foot volume)

![](../Images/png/91951371__Web.png) HEXARAY: POSITION status is shown.

This is a volume control effect.

Normally, this is controlled with an expression pedal connected to the CTL 2, 3/EXP 2 jack.

| Parameter | Value | Explanation |
| --- | --- | --- |
| POSITION | 0–100 | Adjusts the volume. |
| MIN | 0–100 | Sets the volume when the heel of the EXP Pedal is depressed. |
| MAX | 0–100 | Selects the volume when the toe of the EXP Pedal is depressed. |
| CURVE | SLOW1, SLOW2, NORMAL, FAST | You can select how the actual volume changes relative to the amount the pedal is pressed. |

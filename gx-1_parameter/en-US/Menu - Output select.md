# Output select

These settings specify the device (amp) that’s connected to the OUTPUT jacks.

| Value | Explanation |
| --- | --- |
| LINE/PHONES | Choose this setting if you’re using headphones, or if the GX-1 is connected to a keyboard amp, mixer, or digital recorder. |
| JC-120 RETURN | Choose this setting if the unit is connected to the RETURN jack of the Roland JC-120 guitar amp. |
| JC-120 INPUT | Choose this setting if the unit is connected to the guitar input of a JC-120 guitar amp. |
| KATANA-50 POWER AMP IN | Use this setting when connecting to the POWER AMP IN jack of a BOSS KATANA-50 guitar amp. |
| KATANA-100 POWER AMP IN | Use this setting when connecting to the POWER AMP IN jack of a BOSS KATANA-100 guitar amp. |
| SMALL AMP | Use this setting when connecting to a small guitar amp. |
| COMBO SPx2 RETURN | Use this setting when connecting to the RETURN of a combo amp (in which the amp and speakers are in a single unit) equipped with two speakers. |
| COMBO SPx2 INPUT | Use this setting when connecting to the INPUT of a combo amp (in which the amp and speakers are in a single unit) equipped with two speakers. |
| STACK SPx4 RETURN | Use this setting when connecting to the RETURN of a stack-type guitar amp (in which the amp and speakers are separate units).  This assumes that the connected speaker cabinet is equipped with four speakers. |
| STACK SPx4 INPUT | Use this setting when connecting to the input of a stack-type guitar amp (in which the amp and speakers are separate units).  This assumes that the connected speaker cabinet is equipped with four speakers. |

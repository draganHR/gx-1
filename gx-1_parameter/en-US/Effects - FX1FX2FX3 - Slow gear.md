# Slow gear

![](../Images/png/91951371__Web.png)HEXARAY: Reduction meter

This produces a volume-swell effect (“violin-like” sound).

| Parameter | Value | Explanation |
| --- | --- | --- |
| ON/OFF | OFF, ON | Turns this effect on/off. |
| SENS | 0–100 | Adjusts the sensitivity of the humanizer. When it is set to a lower value, the effect of the slow gear can be obtained only with a stronger picking, while no effect is obtained with a weaker picking. When the value is set higher, the effect is obtained even with a weak picking. |
| RISE TIME | 0–100 | Adjusts the time needed for the volume to reach its maximum from the moment you begin picking. |
| LEVEL | 0–100 | Adjusts the volume. |

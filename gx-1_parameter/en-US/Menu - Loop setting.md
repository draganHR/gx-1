# Loop setting

These settings configure how the looper operates.

On the LOOP SETTING screen, the loop automatically turns on, and LOOP CTL is assigned to the [CTL1] (C1) pedal.

| Parameter | Value | Explanation |
| --- | --- | --- |
| LOOP MODE | MONO | Mixes the L/R signals for mono operation. The recording time is 38 seconds. |
|  | STEREO | Operate in stereo. The recording time is 19 seconds. |
| LOOP REC ACTION | RECÓPLAYÓDUB | Operation will switch in the order of Recording Ó Playback Ó Overdubbing. |
|  | RECÓDUBÓPLAY | Operation will switch in the order of Recording Ó Overdubbing Ó Playback. |
| LOOP LEVEL | 1–120 | Sets the volume of the phrase playback. |
